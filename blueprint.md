
# **프로젝트 블루프린트: 명품시계 매입 사이트**

## **1. 프로젝트 개요**

revu.net을 벤치마킹하여, 사용자에게 신뢰감을 주는 전문적인 명품시계 매입 웹사이트를 구축합니다. 현대적이고 세련된 디자인, 직관적인 사용자 경험, 명확한 정보 전달을 목표로 합니다.

## **2. 핵심 기능 및 디자인**

*   **디자인 컨셉:**
    *   **색상 팔레트:** 블랙, 화이트, 골드를 메인으로 사용하여 고급스럽고 신뢰감 있는 이미지를 전달합니다.
    *   **타이포그래피:** 가독성 높은 고딕 계열의 폰트를 사용하여 전문성을 강조합니다.
    *   **레이아웃:** 모바일과 웹에서 모두 최적화된 반응형 디자인을 적용합니다.
*   **주요 섹션:**
    *   **히어로 섹션:** 사이트의 첫인상을 결정하는 부분으로, 강력한 헤드라인과 매입 문의로 바로 이어지는 CTA(Call-to-Action) 버튼을配置합니다.
    *   **매입 프로세스 안내:** '온라인 문의 -> 전문 감정사 상담 -> 제품 발송 -> 최종 검수 및 입금' 등 명확하고 간단한 매입 절차를 시각적으로 안내합니다.
    *   **주요 매입 브랜드:** 롤렉스, 파텍 필립, 오데마 피게 등 주요 명품시계 브랜드 로고를 노출하여 전문성을 어필합니다.
    *   **실시간 매입 현황:** 실제 매입 사례를 보여주는 동적인 섹션 (추후 구현 가능).
    *   **문의하기:** 사용자가 이름, 연락처, 시계 정보, 사진 등을 첨부하여 쉽게 매입을 문의할 수 있는 양식을 제공합니다.
    *   **푸터:** 회사 정보, 소셜 미디어 링크, 저작권 정보를 포함합니다.

## **3. 현재 요청 구현 계획**

1.  **`index.html` 구조 설계:**
    *   시맨틱 HTML 태그 (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)를 사용하여 구조를 잡습니다.
    *   각 섹션(히어로, 프로세스, 브랜드, 문의)에 필요한 HTML 요소를 배치합니다.
2.  **`style.css` 스타일링:**
    *   선택된 색상 팔레트와 폰트를 적용합니다.
    *   각 섹션의 레이아웃과 디자인을 구현합니다.
    *   버튼, 폼 등 인터랙티브 요소에 호버 효과 등 동적인 스타일을 추가합니다.
    *   미디어 쿼리를 사용하여 반응형 디자인을 완성합니다.
3.  **`main.js` 기능 추가:**
    *   네비게이션 메뉴의 부드러운 스크롤 기능을 구현합니다.
    *   (선택) 폼 제출 시 간단한 유효성 검사를 추가합니다.
